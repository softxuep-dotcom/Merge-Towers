# 集成塔：15 技能池与独立升级表（第二轮平衡稿）

> 本文只讨论玩法与数值框架，不修改代码。  
> 参考：`HordeSpark/Survivors2` 的直接技能制、规模 / 强度 / 特性模块成长，以及投射物、场地和 VFX 预算思路。

## 1. 核心结论

- 场上只有 **1 座集成塔**。
- 技能池共 **15 个特效分支**：火、水、电、光、毒各 3 个。
- 一局最多装备 **5 个不同技能**，元素数量不受限制；允许三火两冰、三毒一光一电等组合。
- 集成塔只有 **1 颗主弹道**。技能挂接在主弹的发射、命中、击杀和持续场地区间上。
- 单次主弹最多产生 **3 个可见派生弹道**；电链、光线使用瞬时线段，不占长期投射物。
- 每个已装备技能独立拥有 Lv1–Lv8 和自己的模块等级。
- 集成塔整局可成功移动 **2 次**；无效拖动不消耗次数。

## 2. 一次攻击如何结算

```text
集成塔寻找目标并发射 1 颗主弹
        ↓
【发射阶段】水镜分裂、弹道外观、射速类效果
        ↓
【命中阶段】爆裂、熔核、冰牢、连锁、酸蚀、裁决
        ↓
【后续阶段】碎片、折射光线、辉光爆发、瘟疫传播、孢种爆发
        ↓
【场地阶段】熔岩轨迹、暴风雪、磁暴场、毒池
```

同一发主弹可以触发全部 5 个已装备技能，但视觉遵守以下规则：

1. 主弹始终只有 1 颗。
2. 派生实体弹道按优先级只显示前 3 颗；超出部分仍结算伤害，改用瞬时闪光表现。
3. 同类场地重叠时刷新持续时间并合并范围，不重复创建完整特效。
4. 同时播放的重型命中特效最多 2 个，其余合并为一次复合爆点。

## 3. 怎么获得和升级技能

### 3.1 共享经验与改造点

- 敌人死亡和过波为集成塔提供 **共享经验**。
- 经验条充满时获得 1 个 **改造点**。
- 五个技能不各自吃经验；玩家把共享改造点自由分配给任意技能。
- 金币不参与技能等级，金币的后续用途另行审核。

### 3.2 点击集成塔后的操作

拥有改造点时点击集成塔，显示 5 个技能槽：

- 点击空槽：选择元素，再从该元素 3 个技能中选择 1 个；消耗 1 点，技能以 Lv1 装入。
- 点击已有技能：显示该技能的 3 张模块卡；消耗 1 点，技能提升 1 级。
- 已装备技能不能重复选择。
- 装满 5 个技能后，空槽入口关闭，只能升级现有技能。

建议第一局开场免费选择第一个技能，让玩家在第一波就能看到构筑效果。其余四个技能由改造点装配，玩家可以选择尽快补满，也可以先专精已有技能。

### 3.3 单个技能的 Lv1–Lv8

| 技能等级 | 操作 |
|---|---|
| Lv1 | 装配技能，立即获得基础效果 |
| Lv1→Lv2 | 规模 / 强度 / 特性三选一 |
| Lv2→Lv3 | 规模 / 强度 / 特性三选一 |
| Lv3→Lv4 | 规模 / 强度 / 特性三选一 |
| Lv4→Lv5 | 规模 / 强度 / 特性三选一 |
| Lv5→Lv6 | 规模 / 强度 / 特性三选一 |
| Lv6→Lv7 | 规模 / 强度 / 特性三选一 |
| Lv7→Lv8 | 规模终极 / 强度终极 / 特性终极三选一 |

普通模块可以重复选择。技能到 Lv7 时，规模、强度、特性累计获得 6 个 Rank；例如 2/2/2、4/1/1、6/0/0 都合法。Lv8 再选择一个终极变异，因此每一级都有明确决策。

### 3.4 三类升级卡的统一含义

| 模块 | 主要改变 | 性能约束 |
|---|---|---|
| **规模** | 范围、目标数、碎片数、连锁数、场地尺寸 | 通常由奇数 Rank 先扩范围、偶数 Rank 再增加目标或派生物；可见派生弹道全局仍最多 3 个 |
| **强度** | 命中、持续、碎片、光线等技能伤害 | 每 Rank 使该技能全部伤害 +20%；第 2/4/6 Rank 再强化一次该技能的核心命中 |
| **特性** | 概率、持续时间、控制、触发周期、印记保留等专属机制 | 只使用玩家可读的整数档位，不再出现“−0.2 次攻击”一类隐藏累积 |

文中 `D` 表示集成塔本次主弹的基础伤害。所有百分比先作为审核基准，最终以模拟和实测调整。

### 3.5 第二轮筛查结论

| 处理 | 技能 / 卡片 | 原因 | 新方向 |
|---|---|---|---|
| 删除旧机制并重做 | 圣辉共鸣 | 隐藏的全局增伤会在数值高时变成必选，数值低时又完全无感 | 改为可见的 **辉光蓄能**，由塔身蓄能、靠其他技能充入色彩后主动爆发 |
| 删除旧机制并重做 | 腐蚀毒液 | 对低甲怪近乎无用，对高甲 Boss 又容易变成必选工具人 | 改为 **酸蚀破甲**，3 层必定破裂；无甲敌人也会受到破裂伤害 |
| 删除旧机制并重做 | 孢子增殖 | 与瘟疫毒池同为“派生物 + 小毒场 + 传播”，定位重复且增加场地压力 | 改为 **寄生孢种**，附着后爆发或死亡转移，不再生成毒雾 |
| 大改 | 熔核过载 | 原版又有周期弹、又有震荡范围，与爆裂弹头重叠 | 改为同一目标叠热量的单体 / Boss 专精技能 |
| 大改 | 最终审判 | 原版主要是血线判定，被动、低可见，普通卡在前期也偏弱 | 加入三层审判印记和可见光击，处决只是附属收益 |
| 数值结构重排 | 规模卡 | “第 1 Rank 就 +1 弹 / +1 目标”经常比另外两卡强一倍 | 统一把增量目标放到第 2/4/6 Rank，奇数 Rank 提升范围与搜索能力 |
| 机制增强 | 强度卡 | 单纯 +20% 不鸡肋，但连续选择时决策感不足 | 保留稳定增伤，并在第 2/4/6 Rank 加入核心命中的小里程碑 |
| 可读性重做 | 特性卡 | 小数周期、隐藏冷却返还难判断价值 | 改成明确的概率、层数、持续时间或“每第 N 次”档位 |

三种卡的单 Rank 目标收益都控制在约 **18%–28%**。规模卡偏清群，强度卡偏稳定输出，特性卡偏构筑与控制；任何一种卡都不能只在极少数敌人或特定关卡才生效。

## 9. 技能组合示例

### 9.1 三火两冰

`爆裂弹头 + 熔岩轨迹 + 熔核过载 + 暴风雪 + 水镜碎矢`

- 一颗主弹沿路留下火带。
- 命中后结算爆裂与熔核。
- 命中位置生成暴风雪。
- 满足周期时最多派生 3 枚可见碎片或水镜箭，其余效果用瞬时闪光结算。
- 定位：强范围、强持续、控制稳定，但缺少易伤与远距离连锁。

### 9.2 三毒一光一电

`瘟疫毒池 + 酸蚀破甲 + 寄生孢种 + 辉光蓄能 + 连锁审判`

- 酸蚀叠满后破甲并爆发，低甲敌人也不会浪费该技能。
- 毒池负责持续覆盖，孢种负责延迟爆发和死亡转移，不再重复铺毒雾。
- 中毒死亡产生传播。
- 辉光利用其他技能触发积累色彩充能，电链负责快速补刀。
- 定位：启动较慢，但敌人密度越高越强。

### 9.3 五元素均衡

`熔核过载 + 冰霜牢笼 + 风暴眼 + 最终审判 + 酸蚀破甲`

- 酸蚀提供破裂与易伤，冰牢控制，雷印和熔核提供爆发，审判完成收割。
- 没有大量派生弹道，性能成本最低。
- 定位：Boss 和精英稳定，但清理超大虫群不如纯范围构筑。

## 10. 性能预算

| 项目 | 建议硬上限 |
|---|---:|
| 同时存在的主弹 | 1 |
| 单次主弹产生的可见派生弹道 | 3 |
| 场上长期实体投射物 | 12 |
| 同时存在的逻辑场地 | 6 |
| 同技能完整场地材质 | 2，更多只刷新或合并 |
| 同时播放的重型命中特效 | 2 |
| 中低端设备活跃粒子 | 150–220 |
| 高端设备活跃粒子 | 250–350 |

性能降级只影响画面，不影响玩法：超预算时依次减少尾迹粒子、合并命中特效、隐藏超额派生弹道、降低场地材质层数，但所有伤害、控制与传播仍正常结算。

## 11. 待审核项

1. 是否采用“共享经验 → 改造点”，而不是五条独立经验条。

3. 玩家是否可以一直专精一个技能、暂不装满五槽；本文允许这样做。
4. 单技能是否继续保留 Lv8；本文按 Lv8 设计。
